多类型价差关注列表

Type A：Binance 现货+永续同所对冲，资金费 |FR| ≥ 0.3% 的高资费标的；同一保证金账户爆仓风险最低，收益来自价差周期波动，但需付资金费（除非结算前后滚仓）。
Type B：任意两家永续，|a-b|/min > 1%；套利方向是价差收敛。资金费过滤默认使用“净成本模式”：按“高价做空/低价做多”方向计算两腿净资金费亏损，要求亏损 ≤ 0.1%（允许盈利，不限制单边资金费绝对值；可通过环境变量切回旧的区间模式）。
Type C：现货低于任一期货 > 1%，且两家期货资金费 ≥ -0.1%；做法多现货、空期货，收益主要来自正向基差回归，同时控制资金费不异常、尽量降低付费。

指标释义（主要适用于 Type A）：Spread% = (现货-永续)/现货；均值/基线用近 1h 数据，基线=均值-1.5σ；区间（1h/12h）为高低差；斜率看近 3 分钟涨跌；穿越次数看 3h 内相对 30m 中线的翻转；Drift=|当前-中线|/1h 区间。
信号（Type A）：C1=价差低于基线且低于绝对阈值(5%)，且 3min 斜率≤0；C2=波动/区间达标且穿越≥2且 Drift≤0.3；必须 C1 & C2 同时满足。Type B/C 会展示回归预测(240m)的 SIG（用于参考）；实盘最终是否开仓以 live trading 的订单簿验算口径为准。

指标说明（基于后端实际计算）

Spread%： (现货-永续)/现货，取最新 1m 点；均值/标准差用最近 60 个点算 μ 与 σ，基线=μ-1.5σ（用于信号计算），波动率=σ。
区间 (1h/12h)：对应窗口的最高-最低（剔除资金费前后 1 分钟），区间越大说明越活跃。
斜率：近 3 分钟涨跌幅 (末值-首值)/3；中线：15 点均线，描述短期中枢。
穿越/Drift：3h 内价差相对 30m 中线的翻转次数（判断是否在更长周期里来回波动）；Drift=|当前-中线|/1h 区间，偏移越小越接近震荡。
OI / 保险金：OI=未平仓合约量（合约张数折算的名义价值），反映场内持仓体量；保险金=该所用于穿仓兜底的基金规模，越大意味着极端行情下被动平仓的缓冲能力越强。
历史资金费：取该交易所最近 ≤24 次资金费，输出“最新/均值/标准差 (样本数)”用于对比当前水平与波动，例如 “-0.110% / -0.143% / -0.086% (12)”。
开仓 (C1 & C2 必须同时满足)
C1：价差低于基线且低于绝对阈值 5%，且 3min 斜率≤0；
C2：波动≥0.5%，区间1h≥1%，区间12h≥1.5%，穿越≥2，Drift≤0.3。
平仓信号
TP：用 120min 均值/波动，阈值=μ120+1.2σ120，当前是近20min 局部高点且斜率<0，冷静期15min。资费前落袋：距结算≤30min 且价差 ≥ (μ60+0.5σ60) 也标记 TP。SL：连续2个点低于 -(阈值+0.5%) 且斜率<0；Funding：距离结算 ≤5 分钟提示离场（按交易所的实际周期/下一次结算时间绘制）。
图表：近 12 小时价差；蓝线=价差，绿虚线=15m 中线，黄虚线=基线；青色散点=开仓，绿色=TP，红色=SL，紫色=资金费提醒；橙色虚线=资金费结算。

v2 因子说明（定量定义）

序列：spread_t = ln(P_short/P_long)（两腿 1min close，对齐容忍 ≤90s，最多 forward-fill 2 bars）；funding_edge_t = FR_short − FR_long（若某腿为 spot，则 funding=0）。
收益/斜率：spread_ret_60m = spread_t − spread_{t−60m}；spread_slope_240m = OLS 斜率（窗口 241 个点，x=分钟索引，y=spread）。
波动/形态：spread_vol_30m = std(Δspread) over 30m；spread_zscore_4h = (spread_t−mean_4h)/std_4h；exhaustion=1 当 |slope_5|<|slope_30| 且 ret_5 与 ret_30 反向。
资金费：funding_edge_short_minus_long = funding_edge_t；carry_net_cost_horizon_240m = funding_edge_t × (240 / funding_interval_min)。
质量门槛：missing_rate≤max_missing_ratio（默认 0.2），缺失来自 1min K 对齐；细节见 factors_v2_meta.short_quality/long_quality。
